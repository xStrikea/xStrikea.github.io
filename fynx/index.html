Fynx Notify

Fynx Notify — Client-side Notification System

A modern, black-themed notification GUI for Roblox. Client-only, lightweight, and fully customizable.

Example

Client-only

Notifications are visible only to you; fully local without server replication.

Stackable Alerts

Multiple notifications auto-stack and slide smoothly above your tools bar.

Custom Actions

Add buttons with callbacks for interactive notifications.

Modern UI

Black theme with sleek animations and adjustable typography.

Example

Paste the script below into your executor.


if not _G.FynxNotify then
    loadstring(game:HttpGet("https://example.com/fynx_notify_core.lua"))()
end

task.wait(2)

_G.FynxNotify:Notify({
    Title = "TEST",
    Text = "TEST-001",
    Duration = 4,
    Icon = "rbxassetid://6031071053",
    SoundId = "rbxassetid://12222132"
})

task.delay(5, function()
    _G.FynxNotify:Success("TEST-002", "TEST-003", 3)
end)

task.delay(8, function()
    _G.FynxNotify:Notify({
        Title = "TEST-004",
        Text = "TEST-005",
        Duration = 6,
        ActionText = "TEST-006",
        Callback = function()
            print("TEST-007")
        end
    })
end)

* Ensure HTTP requests are enabled in your executor.
